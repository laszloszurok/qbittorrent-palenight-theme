# qbittorrent palenight theme

A dark theme for qbittorrent.

## How to modify

Edit stylesheet.qss and compile resources.qrc with Qt Resource Compiler (rcc).

For example:
```
rcc resources.qrc -o your_theme_name.qbtheme -binary
```
